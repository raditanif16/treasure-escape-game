# Treasure Escape

Treasure Escape is a 2D platformer game developed using Python and Pygame Zero.

## Features
- 3 Levels (Morning, Sunset, Night)
- Animated Player and Enemy
- Treasure Collection System
- Background Music and Sound Effects
- Game Over and Win Screen
- Cloud and Confetti Effects

## Controls
← → : Move
SPACE / ↑ : Jump

## Requirements
- Python 3.x
- Pygame Zero

## Run
pgzrun main.py
